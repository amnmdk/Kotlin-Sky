 Kotlin-sky

## Description
Kotlin-sky est une application Android développée en Kotlin qui permet d'afficher la météo (fictive) pour une sélection de grandes villes. Cette application est un projet pédagogique réalisé dans le cadre du cours de développement mobile Android.

---

## Configuration du Projet

### Environnement de développement :
- **IDE** : Android Studio
- **Langage** : Kotlin
- **Build System** : Gradle (Kotlin DSL)
- **Minimum SDK** : 21 (Android 5.0)
- **Target SDK** : 28 (Android 9.0 - Pie)
- **Compile SDK** : 28 (Android 9.0 - Pie)

---

## Dépendances principales :
- `androidx.appcompat:appcompat:1.6.1`
- `androidx.constraintlayout:constraintlayout:2.1.4`
- `androidx.core:core-ktx:1.12.0`
- `com.google.android.material:material:1.9.0`
